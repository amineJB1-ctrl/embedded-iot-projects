# Industrial Lifting System

Lifting mechanism using Arduino Uno with HC-SR04 ultrasonic 
sensor for distance measurement and LED bar graph for level 
visualization.

## ⚙️ System Description

The system uses an HC-SR04 ultrasonic sensor to measure the distance between the sensor and an object, simulating the lifting level.

The measured distance is processed by the Arduino Uno and mapped to a range of LEDs. A LED bar graph is used to visually represent the level:

Low distance → fewer LEDs ON
High distance → more LEDs ON

This provides a real-time visual indication of the lifting level, similar to industrial monitoring systems.

## Objective
Simulate an industrial lifting process by measuring distance (level) and displaying it visually using LEDs.

## Components
- Arduino Uno
- HC-SR04 ultrasonic sensor
- LED bar graph
