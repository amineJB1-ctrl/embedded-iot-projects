# Solar Panel Tilt Control System

Arduino-based system for controlling the tilt angle of solar panels using push buttons, a servo motor, and a 7-segment display.

## ⚙️ System Description

The system allows the user to adjust the tilt angle of a solar panel using two push buttons:

* One button increases the angle.
* The other button decreases the angle.

A servo motor is used to change the inclination of the panel based on the selected value. The current angle level is displayed on a 7-segment display, providing a clear visual feedback to the user.

Each value corresponds to a specific tilt position, allowing step-by-step control of the panel orientation. This helps optimize the angle of the solar panel for better energy capture.

## Objective

Adjust the tilt angle of solar panels manually using buttons to improve energy efficiency and optimize solar energy capture.

## Components

Arduino Uno
Servo motor
7-segment display
Push buttons
